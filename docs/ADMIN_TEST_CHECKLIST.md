# 관리자 페이지 테스트 체크리스트

- `cfv47@naver.com` 계정으로 로그인 시 `/admin`과 `/admin/dashboard` 접근 가능 확인
- 다른 이메일 계정으로 로그인 시 "관리자 권한이 없습니다." 표시 확인
- 로그인하지 않은 상태에서 `/admin` 접근 시 로그인 화면으로 이동하는지 확인
- `NEXT_PUBLIC_ADMIN_EMAILS=cfv47@naver.com,example@naver.com` 설정 시 추가 이메일도 관리자 UI에서 허용되는지 확인
- Firestore Rules 배포 후 관리자 이메일이 아닌 계정이 `users`, `payments`, `enrollments`, `certificates` 목록을 직접 조회할 수 없는지 확인
- 대시보드에서 전체 회원 수, 결제 건수, 총 결제금액, 오늘/이번 달 결제금액, 수료증 발급 건수가 표시되는지 확인
- 회원 관리에서 성명, 이메일, 휴대전화번호, 사용자 ID 검색이 동작하는지 확인
- 회원 상세에서 결제내역 수, 수강권 수, 수료증 보기, 예상 환불 정보, 관리자 메모 저장이 가능한지 확인
- 결제 관리에서 최신순 정렬, 결제상태 필터, 오늘/이번 달 결제 필터가 동작하는지 확인
- 결제 상세에서 PG 원본 응답이 표시되고 Secret Key 같은 민감정보가 노출되지 않는지 확인
- 수강권 관리에서 accessStatus, 진행률, completedLessons, 만료일, 남은 수강일이 표시되는지 확인
- 수료증 관리에서 발급번호 검색, 수료증 보기/인쇄 링크가 동작하는지 확인
- 환불 관리에서 미수강 강의 수 x 11,000원 기준 환불액과 환불 불가 사유가 맞는지 확인
- 수료증 발급 완료, 5강 전체 수강 완료, 수강기간 만료, 정상 결제 아님 상태가 환불 불가로 표시되는지 확인
- 강의 관리에서 1-5강 lessonId, 제목, 영상 ID, 재생시간이 표시되는지 확인
- 시스템 설정에서 관리자 이메일 목록과 운영 설정이 표시되고 Secret Key가 표시되지 않는지 확인
- 관리자 메모 저장 시 일반 사용자 화면에는 메모가 보이지 않는지 확인
- 관리자 메모 저장 시 `adminLogs`에 로그 문서가 생성되는지 확인
- 각 목록에서 CSV 다운로드가 실행되는지 확인
