
# Lotto Number Recommendation Site

## Overview

This is a simple web application that generates and displays random lottery numbers. The goal is to create a visually appealing and easy-to-use tool for users who want a random set of lottery numbers.

## Project Outline

### Style and Design

*   **Layout:** The application will have a centered layout with a clean and modern design.
*   **Colors:** A vibrant color palette will be used to make the application feel energetic and fun.
*   **Typography:** Expressive fonts will be used to create a clear visual hierarchy.
*   **Effects:** Subtle shadows and gradients will be used to add depth and a premium feel to the UI elements.
*   **Theme:** The application supports both light and dark modes.

### Features

*   **Number Generation:** The application will generate a set of 6 unique random numbers between 1 and 45.
*   **Display:** The generated numbers will be displayed in a clear and visually appealing way.
*   **Interactivity:** A button will allow users to generate a new set of numbers at any time.
*   **Theme Toggle:** A switch allows users to toggle between light and dark mode.
